# Stage-ete

Afin de pouvoir lancer le site, vue qu'il n'est pas finalisé, le site est en localhost.

Pour pouvoir acceder au site il faut telecharger WAMP (si système d'exploitation Windows), MAMP (si système d'exploitation MacOs) ou LAMP(si système d'exploitation Linux)
Ensuite mettre le chemin du serveur local au niveau du répertoire ou est situé le site.
Ouvrir PHPmyadmin créer une BDD nommée testStage est y inserer les tables qui sont dans /include/BaseDeDonnees
il vous faudra aussi créer un répertoire uploads situer en dehors du dossier stage-ete
